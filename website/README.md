# antivirus.sh landing page

A simple, modern static landing page for https://github.com/ultra-x-coder/antivirus.sh.

## Files

- `index.html` — page markup
- `styles.css` — responsive visual design, light/dark theme variables, and hero terminal styling
- `script.js` — theme toggle persistence and copy button behavior

## Notes

- The hero title uses `font-size: clamp(46px, 8vw, 82px);`.
- The theme switcher saves the selected theme to `localStorage`.
- The dark theme uses Hyper-inspired black, white, border gray, and magenta/violet accents instead of green tones.

Open `index.html` in a browser or deploy the folder to any static hosting service.
