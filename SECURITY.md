# Security Policy

`antivirus.sh` is a security tool, so we take vulnerabilities in it seriously.

If you find a vulnerability (e.g., a way the script could be abused to escalate
privileges, execute attacker-controlled input, or corrupt a system), please
**do not open a public issue**. Report it privately via GitHub Security
Advisories ("Report a vulnerability" on the repository's Security tab).

You will get a response within 7 days. Fixes are released as fast as possible
and credited unless you prefer otherwise.

Hardening suggestions, false-positive reports and new detection ideas are
welcome as regular issues at https://github.com/TARGET_PLEVEHOLDER/antivirus.sh/issues.
